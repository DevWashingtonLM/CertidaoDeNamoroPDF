Crie Certidões de Namoro em PDF: Um Backend Divertido com Spring Boot e Apache PDFBox

Já pensou em ter uma certidão oficial do seu namoro? Este backend em Java com Spring Boot torna isso possível! Ele funciona como uma API RESTful que recebe os dados de um casal (nomes, data de início, termos do relacionamento) e, de forma mágica, gera um lindo documento PDF personalizado. Com o poder do Apache PDFBox, garantimos que cada certidão tenha seu estilo, com imagens de fundo e informações formatadas com carinho. Perfeito para celebrar o amor de uma forma única e memorável!
